# Todo Everywhere

## Overview

旅行に行く感覚で TODO をこなすアプリです。  
It is an application to do TODO as if you were going on a trip!

## Features

- Next.js
- TailwindCSS
- OpenWetherAPI
- Maps Embed API
- Image Creator from Microsoft Bing
